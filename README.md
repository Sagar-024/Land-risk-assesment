Land Purchase Risk Assessment — a Next.js (App Router + TypeScript + Tailwind) tool that takes a US address and intended use, calls the Mireye geospatial API, and produces a plain-English, evidence-backed report of red flags a land buyer wouldn't see by walking the property (flood zone, wetlands, slope, wildfire exposure, protected land, utility access) with a citation for every finding, plus an explicit "not covered" section (title, liens, HOA, mineral rights).
